# 🔄 Hachimi Auto Checkin

### 🔐 1. PRIVATE_REPO_TOKEN（必须）

Name: PRIVATE_REPO_TOKEN
Value: 你刚生成的 GitHub Token

### 👤 2. HACHIMI_BATCH（必须）

### 单账号格式
账号,密码
### 多账号格式（每行一个）
账号1,密码1

账号2,密码2
### 带 Telegram 通知格式
账号,密码,TG_TOKEN,TG_CHATID

示例：

```
test,123456,123456:ABCDEF,987654321
```
### 📡 3. HY2_PROXY_URL（可选）

### 如果需要 Hysteria2 代理：
```
Name: HY2_PROXY_URL
Value: hysteria2://xxxx
```
